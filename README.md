# Weather App

A simple desktop weather app built with PyQt5. Enter a city name and get the current temperature, an emoji representing conditions, and a short description.

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api).

3. Create a `.env` file in the project root (copy `.env.example` and rename it):
   ```
   OPENWEATHER_API_KEY=your_actual_key_here
   ```

4. Run the app:
   ```
   python weather_app.py
   ```
